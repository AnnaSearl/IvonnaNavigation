# @ivonna/navigation

Ivonna 是一套 `Native` 与 `React Native` 的混合开发框架，支持 iOS 与 Android。

## Installation

```sh
npm install @ivonna/navigation
```

## Usage

```js
import { Navigation } from '@ivonna/navigation';

// ...

Navigation.push('Home');
```
